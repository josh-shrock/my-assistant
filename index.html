Import from Google Calendar
Takes about 60 seconds:
  1. Go to calendar.google.com
  2. Click ⚙ gear → Settings
  3. Click your calendar name in the left sidebar
  4. Scroll to Export calendar → click Export
  5. Unzip the downloaded file to get a .ics file
  6. Return here and click Import below
